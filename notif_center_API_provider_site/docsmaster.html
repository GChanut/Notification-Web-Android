Introduction


Si vous êtes Webmaster et désirez que vos utilisateurs soient connectés de partout à votre site, c'est maintenant possible!
Notif'Center est l'intermédiaire entre le site que vous gérez et ses utilisateurs.
Afin d'intégrer notre service, nous vous conseillons de bien lire cette documentation.

Utilisation de l'API

Verify Credentials


Le verify_credentials permet à votre site d'utiliser notre base de données pour une gestion de compte. (Exemple: J'utilise mon compte Facebook pour avoir un compte Deezer).

Afin d'avoir accès au verify_credentials, il lui faut comme paramètre l'access token de l'utilisateur. Il faut donc pour cela que votre utilisateur se soit préalablement inscrit à Notif'Center.

Cela retourna l'username de l'utilisateur, à vous de le manipuler comme bon vous semble pour votre propre gestion de compte.

Il faudra bien penser à inclure la libraire OAuth précédemment téléchargé dans chaque page
Voici un exemple d'utilisation à télécharger: Demo connexion

Dans config.inc.php mettre la consumers_key et la consumer_secret que vous aviez noté et l'adresse du callback (l'adresse du fichier callback.php contenu dans le dossier).
Nous remarquons dans index.php la ligne:
$content = $connection->get('verify_credentials'); //Elle appel la page verify_credentials de l'API

Il y a très peu de chose à modifier dans le reste des pages. Elles permettent surtout la connexion et les étapes pour l'authentificaiton OAuth.

Information importante


Maintenant qu'une session est ouverte sur votre site grâce à Notif'Center, il suffit d'appeler l'access token à chaque requête que vous ferez seulement si cela concerne Notif'Center.
Pour cela, la variable de session $_SESSION['access_token'] contient comme attribut $access_token['oauth_token'] et $access_token['oauth_token_secret'].

Manage Notification


Le manage notification est le service principal de Notif'Center permettant d'envoyer aux utilisateurs de votre site un contenu que vous aurez choisit !

Exemple: Vous êtes webmaster d'un site de CV en ligne, vous décidez d'utiliser Notif'Center afin d'informer ces utilisateur du nombre de visite qu'a pu avoir leurs CV, ou bien de les informer si quelqu'un a répondu à un poste, ou bien même les deux!
Les simples lignes ci-dessous permettent de mettre à jour les notifs des utilisateurs:

session_start();
$access_token = $_SESSION['access_token'];
$content = '[contenu sous format html]';
header('Location: http://notifcenter.zapto.org/notifcenter/oauth2m/api/manage_notification.php?oauth_token='.$access_token['oauth'].'&content='.$content);

ATTENTION: Il est très important de respecter le format html suivant (aux caractères, aux guillemets à l'espace près!!) afin que vos notifications soient correctement interprétées par notre API.
Vous devez aussi mettre un lien de redirection dans le href. Comme ça, lorsque l'utilisateur cliquera sur la notification, il sera automatiquement redirigé vers le site en question.
Remarque: vous pouvez enlever la balise image. <tr> <td onmouseover='javascript:this.style.background="#aaa"' onmouseout='javascript:this.style.background="" ' > <a href="http://notifcenter.zapto.org/notifcenter/docsmaster.html" target='_blank' style='text-decoration: none; color: #cccccc; display:inline-block'> <div style='margin: 5px 5px 5px 5px'><img src='https://graph.facebook.com/100004247014209/picture' align='absmiddle'/> <h4>Texte de la notification</h4> </div> </a> </td> </tr>
Si vous voulez utiliser le php pour dynamiser les notifications:
$content = "<tr> <td onmouseover='javascript:this.style.background=\"%23aaa\"' onmouseout='javascript:this.style.background=\"\" ' > <a href=\"".$p['link']."\" target='_blank' style='text-decoration: none; color: %23cccccc; display:inline-block'> <div style='margin: 5px 5px 5px 5px'> <img src='https://www.google.com/".$p['from']['id']."/picture' align='absmiddle'/> <h4>".$p['title']."</h4> </div> </a> </td> </tr>";


Ainsi, si nous gardons l'exemple du site avec les CVs en ligne, le Webmaster mettra ce bout de code sur la fonction d'ajout de commentaire, ou sur la fonction qui incrémente le compteur de visite. Et dans le "href" il mettra le lien du CV en question.
